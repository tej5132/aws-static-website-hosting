# AWS Static Website Hosting

**InternID:** CITS7018

## Project Overview

This project demonstrates how to host a static website using Amazon S3.

The website was created using HTML, CSS, and JavaScript and deployed using Amazon S3 Static Website Hosting.

## Technologies Used

- Amazon S3
- HTML5
- CSS3
- JavaScript
- GitHub

## Features

- Responsive landing page
- Static website hosted on Amazon S3
- Publicly accessible through an S3 website endpoint

## Deployment Steps

1. Created an Amazon S3 bucket.
2. Uploaded the website files.
3. Enabled Static Website Hosting.
4. Configured the bucket policy.
5. Disabled Block Public Access for the bucket.
6. Verified the website using the S3 website endpoint.
